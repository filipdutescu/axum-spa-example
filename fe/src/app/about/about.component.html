What is this project?

This project is made with the aim to provide an example of how one can build a web application using a SPA framework (such as Angular) and Axum for the API server.

It is a basic IRC-type web application, which supports creating and joining rooms, in which one can talk to multiple users at a time. History is not preserved, thus, if a user leaves the chat, the messages received up until that point will be lost.
